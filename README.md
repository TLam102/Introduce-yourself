# Website Giới Thiệu Bản Thân

## Mô tả dự án
Dự án này là một trang web cá nhân dùng để giới thiệu thông tin bản thân như họ tên, ngành học, mục tiêu học tập và các kỹ năng cơ bản.  
Trang web được xây dựng nhằm mục đích học tập, rèn luyện kỹ năng lập trình web và làm quen với cách tổ chức một dự án hoàn chỉnh.

## Chức năng
- Hiển thị thông tin cá nhân
- Giới thiệu quá trình học tập
- Trình bày mục tiêu nghề nghiệp
- Giao diện đơn giản, dễ sử dụng
- Hoạt động tốt trên trình duyệt web

## Công nghệ sử dụng
- HTML: Xây dựng cấu trúc trang web
- CSS: Thiết kế giao diện và bố cục
- JavaScript (cơ bản): Tạo một số hiệu ứng đơn giản (nếu có)
- Trình duyệt: Google Chrome, Microsoft Edge

## Hướng dẫn chạy
1. Tải hoặc sao chép thư mục dự án về máy
2. Mở file `index.html`
3. Chạy file bằng trình duyệt web bất kỳ
4. Trang web sẽ hiển thị nội dung giới thiệu bản thân
